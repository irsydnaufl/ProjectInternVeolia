# Preview Excel HR Division

Berdasarkan struktur file Excel HR Division, di bawah akan dijelaskan kegunaan dari setiap sheet

- **Sheet All Kode**<br>
berisi daftar singkatan atau kode unik, misalnya Kode Departemen, Kode Jabatan, Kode Barang ATK, dan Kode Operasional.<br>
![Uploading Sheet All Kode.png…](https://github.com/irsydnaufl/ProjectInternVeolia/blob/main/HR/Sheet/Sheet%20All%20Kode.png)<br>
- **Sheet Data**<br>
Sheet ini berisi Master Data atau sumber informasi utama dalam sistem HR. Terdapat data profil setiap individu seperti Nama, ID Karyawan, Jabatan, Departemen, dan Jenis Kelamin.<br>
![Uploading Sheet Data Karyawan.png…](https://github.com/irsydnaufl/ProjectInternVeolia/blob/main/HR/Sheet/Sheet%20Data%20Karyawan.png)
- **Sheet Kehadiran**<br>
Sheet ini fokus pada pencatatan aktivitas harian dan bulanan karyawan, seperti kehadiran dan keterangan absen (Sakit, Izin, Keperluan, Hamil).<br>
![Uploading Sheet Kehadiran.png…](https://github.com/irsydnaufl/ProjectInternVeolia/blob/main/HR/Sheet/Sheet%20Kehadiran.png)<br>
- **Sheet Inventaris Aset**<br>
Sheet ini berisi daftar barang milik perusahaan yang disediakan untuk karyawan, seperti ATK dan Operasional.<br>
![Uploading Sheet Inventaris Aset.png…](https://github.com/irsydnaufl/ProjectInternVeolia/blob/main/HR/Sheet/Sheet%20Inventaris%20Aset.png)<br>
- **Sheet Pivot Table**<br>
Sheet ini merupakan bagian analisis atau ringkasan dari semua data mentah.<br>
![Uploading Sheet Pivot Table.png…](https://github.com/irsydnaufl/ProjectInternVeolia/blob/main/HR/Sheet/Sheet%20Pivot%20Table.png)<br>
- **Sheet Dashboard**<br>
Menampilkan dashboard atau visualisasi dari data yang ada.<br>
![Uploading Insight HR Dashboard .png…](https://github.com/irsydnaufl/ProjectInternVeolia/blob/main/HR/Insight%20HR%20Dashboard%20.png)
